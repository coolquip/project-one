CLONING GITHUB REPO

  1. Open Git Bash.
  2. Change the current working directory to the location where you want the cloned directory to be made.
  3. Type git clone, and then paste the URL you copied in Step 2.
  4. $ git clone https://github.com/YOUR-USERNAME/YOUR-REPOSITORY
  5. Press Enter. Your local clone will be created.

PUSHING GITHUB REPO

  1. As an example, you usually run git push origin master to push your local changes to your online repository.
